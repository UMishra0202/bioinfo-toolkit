# 🧬 Bioinfo Toolkit

A Python-based bioinformatics toolkit for basic sequence analysis, including FASTA parsing and GC content calculation.

---

## 🚀 Features

- Parse FASTA files
- Calculate GC content of sequences
- Command-line interface (CLI) support
- Modular code structure (`main.py` + `utils.py`)

---

## 📁 Project Structure

bioinfo-toolkit/
│── src/
│   ├── main.py        # CLI and execution
│   └── utils.py       # Core sequence functions
│
│── data/
│   └── sample.fasta   # Example input file
│
│── README.md
